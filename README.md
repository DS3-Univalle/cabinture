# [CABINTURE]([http://demos.creative-tim.com/soft-ui-dashboard-react/#/dashboard?ref=readme-sudr](https://stamayo.atlassian.net/wiki/spaces/DS/overview))
Cabinture es una aplicación que ofrece a los usuarios la oportunidad de explorar un extenso catálogo de cabañas y realizar reservas desde la comodidad de su hogar. Además, brinda a los propietarios de cabañas la posibilidad de listar sus propiedades para alquilarlas. Con cabinture, encontrar y reservar la cabaña perfecta es más fácil que nunca.

## Terminal Commands

1. Download and Install NodeJs LTS version from [NodeJs Official Page](https://nodejs.org/en/download/).
2. Navigate to the root ./ directory of the product and run `yarn install` or `npm install` to install our local dependencies.
